# Atlas AI 

## 1. Project Goal

Atlas AI is an intelligent career mentorship platform designed to help aspiring software engineers and AI professionals become industry-ready through personalized guidance.

Unlike a traditional chatbot, Atlas acts as a long-term AI mentor. It analyzes a user's current skills, experience, learning progress, and career goals to generate customized learning roadmaps, identify skill gaps, recommend projects, and track overall progress. Its purpose is to eliminate uncertainty in the learning journey by providing clear, actionable guidance tailored to each individual.

The long-term vision is to create an adaptive mentor that learns from user behavior and continuously improves its recommendations based on the user's progress, learning style, and career objectives.

---

# 2. Target Users

Atlas is initially designed for:

* Engineering students
* Computer Science students
* Self-taught programmers
* Career switchers pursuing software engineering or AI
* Individuals preparing for internships or full-time technical roles

Future versions may expand to support professionals already working in the software industry.

---

# 3. Minimum Viable Product (MVP)

Version 0.1 of Atlas will include the following core features:

### 3.1 User Onboarding

Atlas collects essential information about the user, including:

* Name
* Career goal
* Current education
* Current skill set
* Skill proficiency
* Short-term goals
* Long-term goals
* Daily or weekly study availability
* Learning preferences
* Internship or job search status
* Resume upload
* GitHub username (optional)

---

### 3.2 Resume Analysis

Users can upload a resume in PDF format.

Atlas extracts useful information such as:

* Skills
* Education
* Projects
* Work experience
* Certifications

This reduces manual data entry and creates an initial user profile.

---

### 3.3 Skill Gap Analysis

Atlas compares the user's current skills with the requirements of their chosen career path.

Example:

Goal:
Machine Learning Engineer

Current Skills:

* Python
* NumPy
* Pandas

Missing Skills:

* SQL
* Docker
* AWS
* MLOps

Atlas prioritizes the missing skills based on importance.

---

### 3.4 Personalized Learning Roadmap

Using the user's profile and available study time, Atlas generates a personalized learning roadmap.

The roadmap includes:

* Skills to learn
* Recommended order
* Suggested resources
* Recommended projects
* Estimated completion timeline

---

### 3.5 Dashboard

The dashboard provides a summary of the user's progress, including:

* Current career goal
* Skill progress
* Missing skills
* Learning roadmap
* Next recommended task
* Overall job-readiness score (initial heuristic)

---

# 4. Features Excluded from Version 0.1

To keep the MVP focused, the following features will not be included in the first release:

* AI chatbot conversations
* Voice interaction
* Interview simulation
* Coding challenge evaluation
* Job application tracking
* LinkedIn integration
* Email integration
* Calendar synchronization
* Team collaboration
* Mobile application
* Advanced analytics
* Multi-user collaboration

These features are planned for future versions.

---

# 5. Success Criteria

Atlas Version 0.1 will be considered successful if it can:

* Successfully onboard a new user.
* Accept and analyze a resume.
* Build a structured user profile.
* Identify missing skills based on a selected career goal.
* Generate a personalized learning roadmap.
* Display progress through an intuitive dashboard.
* Recommend the next learning task based on the user's current progress.

---

# 6. Guiding Principles

Atlas is built around the following principles:

* Personalization over generic advice.
* Action over passive learning.
* Continuous improvement through progress tracking.
* Simple and intuitive user experience.
* Recommendations backed by user data.
* Growth through mentorship rather than one-time assistance.

---

# 7. Product Vision

Atlas aims to become more than a learning platform. Its goal is to function as a lifelong AI mentor that adapts to each user's goals, strengths, weaknesses, and learning habits.

Rather than simply answering questions, Atlas will guide users throughout their engineering journey, helping them make informed decisions, stay accountable, build meaningful projects, and ultimately become industry-ready professionals.
